

# 🏨 Hotel Janro – Hotel Management System

**Hotel Janro** is a full-stack **Hotel Management System** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.
The system is designed to simplify hotel operations by providing multiple role-based dashboards and management modules for different hotel services.

This platform helps manage **rooms, reservations, restaurant services, wedding halls, pools, billing, and customer interactions** efficiently.

---

## 🚀 Project Overview

Hotel Janro provides a centralized system where different staff members and customers can access services through dedicated dashboards.

The system improves hotel workflow by:

* Managing bookings and reservations
* Handling hotel facilities
* Processing customer payments
* Organizing hotel service operations

---

## 🎯 Role-Based Dashboards

The system includes **four main dashboards** designed for different users.

### 👨‍💼 Admin Dashboard

The Admin has full control over the system.

Features:

* Manage users and staff
* Monitor hotel activities
* View system reports
* Manage hotel services

---

### 🛎 Reception Dashboard

Used by the reception staff to handle guest interactions.

Features:

* Guest check-in and check-out
* Reservation management
* Customer information tracking
* Room availability monitoring

---

### 👤 Customer Dashboard

Customers can manage their own bookings and services.

Features:

* View and manage reservations
* Access hotel services
* View booking history

---

### 💰 Cashier Dashboard

Handles hotel payments and billing operations.

Features:

* Payment processing
* Bill generation
* Transaction records

---

## ⚙️ Installation Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/hotel-janro.git
```

---

### 2️⃣ Navigate to the Project Folder

```bash
cd hotel-janro
```

---

### 3️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

### 4️⃣ Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

### 5️⃣ Configure Environment Variables

Create a `.env` file inside the **backend folder** and add your MongoDB connection:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

---

## ▶️ Running the Application

### Start Backend Server

```bash
npm start
```

---

### Start Frontend Application

```bash
npm start
```

The application will run on:

```
Frontend : http://localhost:3000
Backend  : http://localhost:5000
```
